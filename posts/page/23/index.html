中国AI的"电力武器"：Token价格只有美国的1/7 中国AI模型调用量暴增127%，首次超越美国 上个月发生了一件大事。中国AI模型的API调用量，同比暴涨127%，第一次在调用量上超过了美国。 这意味着什么？ 以前大家总觉得美国AI更强。但现在看来，中国AI正在悄悄超车。 价格才是真正的杀手锏 业内人都知道，中国AI的Token价格，只有美国的七分之一。 这是什么概念？ 同样花100美元，美国只能买1百万Token，中国能买700万。 这价格差，太夸张了。 东数西算：电力成本的秘密 中国凭啥能这么便宜？ 答案藏在"东数西算"这个国家战略里。 东部做计算，西部供电力。 西部的水电、风电、太阳能，成本只有东部的三分之一。 再加上特高压电网技术，电力从西部送到东部，损耗极低。 所以中国AI的成本结构，天生就有优势。 一场关于"电力"的战争 AI本质上是电力游戏。 训练一个模型，需要多少电？ 推理一次，需要多少电？ 成本怎么算？ 这些问题决定了AI公司能不能赚钱。 中国现在打的，就是"电力成本"这张牌。 电网股的机会 逻辑很清楚： AI调用量暴增 → 电力需求暴增 → 特高压、配电设备、清洁能源站上风口。 这不是猜测，这是正在发生的事情。 3月9日

为什么伊朗战争不会轻易结束？历史告诉你答案 特朗普说"很快结束"，伊朗说"战事由我们决定"。这场冲突会打多久？看看历史就知道了。 特朗普想结束，伊朗不答应 3月10日，美以对伊朗的军事打击进入第11天。 特朗普在白宫说了一句话：“很快会结束。” 同一天，伊朗伊斯兰革命卫队发言人纳伊尼回应：“战事的结束，将由伊朗决定。” 两句话，一个想停，一个要打。 这场战争，没那么容易收场。 先看看伊朗打了什么 11天，伊朗交出了一份答卷： 600次导弹袭击 2600次无人机行动 **200+**个美以目标被摧毁 7部尖端雷达被打掉（包括卡塔尔"沙漠之眼"） 以色列最大发电站被炸，特拉维夫全面停电 这不是被动挨打，这是有组织的反击。 纳伊尼还说了一句话：伊朗有能力以当前节奏持续6个月。 为什么伊朗这么硬？ 第一，这是"存亡之战" 伊朗给这场战争定的性很清楚：存亡之战。 什么叫存亡之战？ 就是输了政权就没了，国家就完了。 这种仗，没人会轻易投降。 看看历史就知道—— 两伊战争（1980-1988），伊朗打了8年。 当时伊拉克有美国、苏联、海湾国家支持，伊朗几乎孤立无援。 但伊朗就是不投降，甚至用上万少年用身体排地雷。 最后呢？ 双方都打不动了，才在联合国调停下停火。 8年，伊朗挺过来了。 第二，伊朗有"代理人网络" 伊朗不是一个人在打。 它身后有： 真主党（黎巴嫩，已加入战局） 胡塞武装（也门，随时可能出手） 伊拉克民兵（已开始袭击美军基地） 叙利亚（伊朗的盟友） 3月2日，真主党已经参战。 这意味着以色列要面对南北两线。 战争一旦扩大，美以的压力会指数级上升。 第三，核设施没完全炸掉 美以的目标之一是摧毁伊朗核能力。 但美军自己的情报显示：核设施仅倒退了几个月。 伊朗的核设施藏在地下，有的深达90米。 空袭能炸毁地面建筑，但很难完全摧毁地下核心。 只要核材料还在，核科学家还在，伊朗的核计划就能恢复。 第四，美以内部有分歧 特朗普想停，内塔尼亚胡想打。 这不是秘密。 特朗普的原话：“结束对伊朗的战争，将是一个与以色列’共同作出的决定’。” 注意，是"共同作出"，不是"以色列决定"。 说明什么？ 美国不想被以色列拖入长期战争。 但内塔尼亚胡的目标是政权更迭，这需要时间。 3月5日，特朗普甚至要求伊朗**“无条件投降”**。 伊朗总统的回应：“这是幻想。” 历史的教训 两伊战争：8年 1980年，萨达姆以为能速战速决。 结果打了8年，双方损失惨重，谁也没赢。 阿富汗战争：20年 苏联打了10年，撤了。 美国打了20年，也撤了。 阿富汗还是那个阿富汗。 伊拉克战争：8年 2003年，美国以为几个月就能搞定。 结果陷入泥潭8年，花了2万亿美元。 这些战争有一个共同点： 发动者都以为会很快结束。 结果都变成了长期消耗。 ...

{ "cover": { "alt": "Neuromorphic computing chip with brain-inspired neural networks, showing energy-efficient architecture for solving complex mathematical equations" } 类脑芯片竟然能解数学题了 这个消息挺让人意外的. 桑迪亚国家实验室的两位研究员–Brad Theilman 和 Brad Aimone–刚刚在《Nature Machine Intelligence》发了篇论文. 他们证明了类脑计算机能解偏微分方程. 等等,什么情况? 类脑芯片不是模仿大脑的吗?大脑又不擅长做数学计算,这种芯片怎么可能搞定严谨的数学问题? 但事实证明,它能. 而且效果还不错. 为什么这事儿挺重要 偏微分方程听着挺玄乎,其实就是描述物理现象的工具. 天气预报要用它. 飞机设计要用它. 核反应堆模拟也要用它. 传统计算机解这些方程,能耗惊人. 类脑芯片不一样.它的架构更像人脑,处理信息的方式更省电. 对国家安全来说,这意义重大. 大脑其实一直在"算"物理 你打网球的时候,大脑在干嘛? 它在预测球的轨迹. 风速,旋转,弧线–这些物理问题,你的大脑瞬间就"解"出来了. 你不会意识到自己在算方程. 但你确实在算. 类脑芯片就是想模仿这种能力. 下一步:类脑超算? 这个突破可能改变一切. 如果类脑芯片真的能高效处理复杂的数学问题,那我们离第一台类脑超级计算机就不远了. 想想看. 一台超算,能耗只有传统机器的零头. 却能模拟核武器,预测气候,设计新药. 这事儿挺值得期待的. 两位Brad可能自己都没想到,类脑芯片能干这活儿. 科技的发展就是这样. 你以为它在走弯路,结果它开辟了新方向.
