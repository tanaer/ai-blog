给编码 Agent 写 Skills 是门艺术.写得好,效率翻倍;写得不好,agent 迷路.

问题是:**手动写 Skills 太累了**,而且每个仓库都需要定制.

GEPA 刚发布了 **gskill**,一个全自动化的技能学习管线.给定任意 GitHub 仓库,它会**自动生成最适合这个仓库的 Agent Skills**.

最关键的是:**用便宜的小模型学出来的技能,可以直接迁移到 Claude Code 上用.**

## 核心思路
gskill 由两个组件组成:

1. **SWE-smith**:从目标仓库自动生成可验证的 SWE 任务(带测试)
2. **GEPA optimize_anything**:进化式优化循环,自动学习和改进 Skills

流程:
```
仓库 → SWE-smith 生成任务 → GEPA 优化循环迭代 Skills → 输出 SKILL.md

优化循环的原理:从空/基础 Skills 开始,让 Agent 执行任务,收集反馈,然后用更强的 LLM 反思这些结果,提出改进建议,迭代优化.

## 效果有多强?

### 在 Mini-SWE-Agent(gpt-5-mini)上
Jinja, 基线=55%, 加 Skills 后=82%, 提升=+27%
Bleve, 基线=24%, 加 Skills 后=93%, 提升=**+69%**

Bleve 是 Go 项目,说明这套框架不仅限于 Python.

### 迁移到 Claude Code
技能是在 gpt-5-mini 上学的,但可以直接用在 Claude Code 上:

**Bleve(Go 项目)**:
- Claude Haiku 4.5:79.3% → 100%,时间 173s → 142s
- Claude Sonnet 4.5:通过率饱和,但时间显著减少

**Jinja(Python 项目)**:
- Claude Haiku 4.5:93.9% → 98.5%

**关键发现**:你可以用便宜的小模型学习 Skills,然后部署到生产级的 Agent 上.

## 学到的 Skills 长什么样?
以 Bleve 为例,自动学到的 Skills 包含:

```markdown

## Run tests early and iterate from failures
- Start broad: `cd /testbed && go test ./...`
 - Narrow quickly:
 - package: `go test ./path/to/pkg`
 - single test: `go test ./path/to/pkg -run TestName -count=1`
- For panics: follow the stack trace top frame in repo code first.
- For mismatches: use "expected vs got" to locate the producing function.

## Make minimal, reviewable changes
- Change one behavior at a time; rerun the smallest reproducing test after each change.
- Add focused unit tests when coverage is missing.
- Avoid scratch main.go files in repo root.

这些 Skills 是从 Agent 的成功和失败中学到的,针对性极强.

## 应用场景
**场景一:新项目上手**

团队来了新人,要快速上手一个复杂仓库.用 gskill 自动生成这个仓库的 Skills,新人 + AI Agent 都能更快理解项目.

**场景二:批量优化内部仓库**

公司有几十个内部仓库,每个都配 Skills 太贵.用 gskill 自动学习,一次配置,全公司受益.

**场景三:成本优化**

用 gpt-5-mini 学习 Skills(便宜),然后在 Claude Sonnet 4.5 上部署(准确).成本和效果兼得.

## 快速开始
```bash

# 克隆 GEPA
git clone https://github.com/gepa-ai/gepa.git
cd gepa

# 使用 gskill 学习任意仓库的 Skills
python gskill.py --repo pallets/jinja --output skills/jinja/SKILL.md

# 将 Skills 复制到 Claude Code 项目
cp skills/jinja/SKILL.md /path/to/your/project/.claude/skills/jinja/

**一句话**:gskill 让 Skills 从"手工编写"变成"自动学习".用小模型训练,大模型部署,成本降,效果升.
