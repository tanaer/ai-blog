## 发现一个有意思的工具
Residue 把 AI agent 对话和 Git commit 关联起来.AI 帮你改了代码,但你找不到它做了什么?以后不会了.

## 谁能用?解决什么问题?
**场景**:AI 帮你写了代码,提交了,但你不知道它改了啥,以后想回溯也找不到对话记录.

**Residue 做的事**:

1. **跟踪**:你跟 AI agent 聊天,Residue 默默记录会话
2. **关联**:git hook 把待处理的对话和 commit SHA 绑定
3. **上传**:git push 时话会上传到自托管 worker
4. **浏览**:在后台能看到每个 commit 对应的对话

**工作流**:

```
你聊天 → Residue 记录
 ↓
git commit → hook 关联 commit SHA
git push → 会话上传到你的 worker
后台浏览 → 每个 commit 对应的 AI 对话

## 相比现有方案好在哪?
- **自托管**:部署自己的 Cloudflare Worker,数据留在自己基础设施, **可搜索**:对话能被 AI 搜索,不再丢失上下文, **开源免费**:自己部署,不用交月费

用 Cloudflare R2 存对话 + S3 API token.
