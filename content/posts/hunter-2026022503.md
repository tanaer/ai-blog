Go 工具链生态又添新成员。**go-size-analyzer** 是一个分析编译后 Go 二进制文件大小的工具，支持跨平台格式（ELF、Mach-O、PE、WebAssembly），并提供树状图可视化。

最酷的是，它编译成了 WebAssembly，托管在 [gsa.zxilly.dev](https://gsa.zxilly.dev)，你可以直接在浏览器中打开编译好的二进制文件进行分析——无需安装任何东西。

功能亮点：
- **详细分解**：按包、按段展示大小占比
- **多种输出**：text、json、html、svg
- **交互式界面**：Web UI 和终端 UI 都支持
- **Diff 模式**：对比两个二进制文件的大小变化

安装也很简单：
```bash
# Homebrew
brew install go-size-analyzer

# Scoop
scoop install go-size-analyzer

# Go install
go install github.com/Zxilly/go-size-analyzer/cmd/gsa@latest
```

本地使用：
```bash
gsa --web your-binary
```

Simon Willison 试用了这个工具，发现 Datadog 团队用它把 Agent 二进制体积**减少了 77%**。对于微服务和容器化部署场景，这个工具能帮你快速定位体积膨胀的源头。

项目地址：[github.com/Zxilly/go-size-analyzer](https://github.com/Zxilly/go-size-analyzer)
