发现一个有意思的东西:AI Council v2 ️

让多个 AI 人格开会讨论,最后给你一个综合答案.

## 谁能用?
需要做复杂决策的:
- 产品经理做竞品分析, 开发者做技术选型, 任何人想要多角度意见

## 怎么用
开源,浏览器运行,无需后端:

```bash
git clone https://github.com/prijak/Ai-council

# 打开 index.html 就能用
```

输入问题,它会:
1. **Stage I**:所有成员给首次意见(并行)
2. **Stage II**:互相评审匿名回复
3. **Stage III**:Chairman 综合成最终答案

## 有啥特别的?
**35 种内置人格**:
- Think Tank:分析师,战略家, Corporate:CEO,CFO,CTO, Startup:创始人,增长官, Consulting,Editorial,Medical,Legal...

**多 Provider 支持**:
- Ollama(本地)
- OpenAI,Anthropic,Google,Groq
- 可以混用:本地跑 DeepSeek-R1 + 云端 Claude

**三阶段流程**:
不是简单投票,而是"首次意见 → 同行评审 → 综合裁决".

**附加功能**:
- 温度滑块(精确 vs 创意), 追问(保留上下文), 导出 Markdown/PDF, 11 个预建模板
