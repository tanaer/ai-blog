刚发现一个解决 AI Agent 痛点的工具:MailCat ️

## 谁能用?
AI Agent 能浏览网页,写代码,自动化工作流,但遇到"请验证邮箱"就卡住了.

MailCat 给 Agent 一个自己的邮箱:
- 一键创建邮箱,不用注册, 自动提取验证码, 自动提取邮件里的链接

## 怎么用?
```python
import requests

# 创建邮箱
r = requests.post("https://api.mailcat.ai/mailboxes")
data = r.json()["data"]
email = data["email"] # swift-coral-42@mailcat.ai
token = data["token"]

# 检查收件箱
inbox = requests.get(
 "https://api.mailcat.ai/inbox",
 headers={"Authorization": f"Bearer {token}"}
).json()

# 获取验证码(自动提取!)
if inbox["data"]:
 email_response = requests.get(
 f"https://api.mailcat.ai/emails/{inbox['data'][0]['id']}",

 print(f"验证码: {email_response['data']['code']}") # 自动提取!
```

## 好在哪?
- **一键创建**: API 调用即可,无需注册
- **自动提取**: 验证码,链接自动从邮件解析
- **隐私优先**: 邮件 1 小时后自动删除
- **自托管**: 可部署到自己的 Cloudflare(免费)
- **开源**: MIT 许可证
