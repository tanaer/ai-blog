## CSS 不是编程语言？再想想

当你说 "CSS 只是样式表" 时，rebane2001 用行动打了所有人的脸 —— **他用纯 CSS 写了一个完整的 x86 CPU 模拟器**。

这不是玩具，是真正能跑机器码的模拟器。

<!--more-->

## 它是如何工作的

页面上运行的是一个 C 程序，被 GCC 编译成原生 8086 机器码，然后**完全在 CSS 中执行**。

关键点：
- **零 JavaScript 依赖** — 有一个可选的 JS 时钟加速，但 CSS 自己也实现了时钟
- **纯 CSS 时钟** — 使用 CSS 动画 + container queries 实现
- **不需要用户交互** — 不像其他 "hover 才能运行" 的方案，这个全自动

## 技术黑魔法

作者使用了多种 CSS 技巧实现这个不可能的任务：

1. **CSS 变量模拟寄存器** — 用 `--reg-ax` 这样的变量存储 CPU 状态
2. **Container Queries 做时钟** — 利用容器查询实现自动推进
3. **选择器做逻辑门** — 用复杂的选择器实现布尔运算

## 为什么这很重要

这证明了 CSS 的图灵完备性（虽然一直有争议）。更重要的是，它展示了：
- 现代 CSS 的能力边界
- 极客精神的极致表达
- 技术社区永远不缺疯狂创意
