️ 发现一个有意思的工具叫 Vouch

开源项目现在被 AI 垃圾贡献淹没了.cURL 上周 16 小时收到 7 个 AI 生成的假漏洞报告,维护者心态崩了直接关掉 Bug 赏金计划.

Mitchell Hashimoto(Vagrant,Terraform 作者)出手了,做了个 Vouch.

**谁能用?**

- 你的开源项目被 AI 垃圾 PR/Issue 淹没
- 想建立可信贡献者网络的项目
- GitHub 项目一键集成,Actions 自动检查

**怎么用?**

1. 项目里加个 `VOUCHED.td` 文件
2. 配置 GitHub Actions(check-issue,check-pr)
3. 未背书的用户提交 PR?自动关闭
4. 维护者在 Issue 里评论 `vouch @用户名` 就能背书

**优势在哪?**

- 不是黑名单,是**信任网络**:一个项目背过的人,另一个项目也认, 维护者完全控制:只有 write 权限的人能背书, 透明公开:背书列表就是文本文件,随便查

开源以前靠"先信任后验证",现在 AI 把这模型打破了.Vouch 换成"先验证后信任",跟社区打招呼,介绍一下自己就能被背书,跟正常社交没两样.

GitHub: https://github.com/mitchellh/vouch
