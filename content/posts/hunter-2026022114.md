刚发现一个有意思的 CLI 工具:Skill Check ️

## 谁能用?
写 AI Agent 技能文件的.现在 Claude,Cursor,OpenAI Codex,Gemini CLI 都支持 SKILL.md 格式--把常用的指令和资源打包成技能,让 Agent 按需加载.

问题是:技能文件写得好不好?frontmatter 缺字段?描述太短?链接断了?

Skill Check 就是干这个的.

用法:

```bash
npx skill-check .
```

## 有什么特别的?
- **质量评分**: 0-100 分,按 frontmatter,描述,正文,链接,格式加权
- **自动修复**: `--fix` 安全修复格式和元数据问题
- **安全扫描**: 可选集成 agent-scan,检测恶意代码
- **GitHub URL**: 直接扫描远程仓库,不用 clone
- **多格式输出**: text,JSON,SARIF,HTML,GitHub 注释

## 好用的场景
- **CI/CD 集成**:PR 里自动检查技能质量, **技能市场维护**:确保发布的技能符合规范, **团队协作**:统一技能文件风格

# 自动修复
npx skill-check . --fix

# 检查 GitHub 仓库
npx skill-check https://github.com/user/skills-repo

# 生成 HTML 报告
npx skill-check . --format html
