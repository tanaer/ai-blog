刚发现一个有意思的macOS 工具:TabType ⌨️

## 谁能用?
每天重复打相同东西的人:git 分支名,文件路径,npm 脚本,常用的回复模板...

传统文本扩展工具(TextExpander,Espanso)的问题是:**你得手动配置所有内容**.

TabType 的做法:**自动发现项目上下文,零配置**.

用法:在任何应用里输入 `;;`,弹出补全菜单.

## 有什么特别的?
- **项目感知**: 自动识别 git 分支,文件路径,npm 脚本,Makefile 目标,.env 变量
- **作用域搜索**: `;;@git` 查分支,`;;@f` 查文件,`;;@npm` 查脚本
- **上下文切换**: 终端和浏览器显示不同补全,工作项目和副业项目不同补全
- **隐私优先**: 无账号,无遥测,完全本地运行

配合 Claude Code 或 Cursor 用很香:输入 `;;` 直接补全 skill 名称,文档文件路径.

## 和 Espanso 比呢?
- Espanso:免费开源,但需要手动配置 YAML
- TabType:$9.99 一次买断,自动发现 + 可自定义

我试了一下,确实零配置就能用.打开终端,`;;@git` 直接列出所有分支,选一个就插入.
