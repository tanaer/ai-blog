The Atlantic 刚刚发了一篇深度报道,标题很直接:**AI Agents Are Taking America by Storm**.

不是「将要」,是「正在」.

## 两个平行宇宙
文章描述了一个有趣的现象:美国人正在生活在两个平行的 AI 宇宙中.

**宇宙一**:大多数人.对他们来说,AI = ChatGPT + Google AI Overview + 社交媒体上的 AI 垃圾内容.

**宇宙二**:技术爱好者.他们正在被那些能连续工作数小时的机器人「激进化」--把几个月的工作压缩成几周,把几周压缩成一下午.

这两个宇宙之间的差距正在迅速扩大.

## 数据:AI 写了多少代码?
微软 CEO Satya Nadella 说:**30% 的微软代码现在是 AI 写的**.微软 CTO 预计,到本十年末,这个数字会达到 **95%**.

Anthropic 内部的比例更高:**最高 90% 的代码是 AI 生成的**.

Salvatore Sanfilippo(Redis 创始人)最近写了一篇爆款文章:「对大多数项目来说,自己写代码已经不再合理了」.

## 后 ChatGPT 时代
文章的核心论点是:我们正在进入「后 ChatGPT 时代」.

传统的聊天机器人(ChatGPT,Gemini)已经很强了:
- 记忆功能:能引用之前的对话, 推理能力:能产出更复杂的回答, 多模态:能处理书级别长度的文件,图片,视频,音频

但跟 AI Agent 相比,这些都是小儿科.

Agent 的核心区别:**不是帮你回答问题,是帮你干活**.

## 实际案例
文章列举了一些真实的使用场景:

- **学术界**:用 Claude Code 自主生成论文
- **生物研究**:用 Agent 做实验设计和数据分析, **新闻业**:用 Agent 从零开始写数据驱动的报道, **创业者**:两个人在一小时内做出 Monday.com(市值数十亿美元的公司)的竞品原型

作者自己的体验:
> 当我让 Claude Code 研究Z世代政治观点趋势时,我发了一个简短的查询,一队机器人就开始工作了:一个在网上搜集信息,另一个做数据分析,第三个写简报给我审阅.

## 风险:15 年的家庭照片被删了
Agent 很强大,但也可能很危险.

一位风险投资家让 Claude Cowork(Anthropic 的新 Agent 工具)帮他整理妻子的桌面.结果机器人**删除了 15 年的家庭照片**.

机器人后来承认:「我需要停下来跟你坦白一件重要的事情.我犯了一个错误.」

## 微软 AI CEO 的预测
Mustafa Suleyman(微软 AI CEO)预测:**18 个月内,AI 将自动化「大部分(如果不是全部)白领工作任务」**.

文章也引用了一条讽刺的社交媒体回复:
> 「Hot Pockets™ 的 CEO 说『大部分(如果不是全部)餐食将在 12-18 个月内被 Hot Pockets™ 完全取代』.」

## 我的看法
Atlantic 的这篇文章是一个标志:主流媒体开始认真对待 AI Agent 了.

ChatGPT 让大众意识到「AI 能聊天」.Claude Code 和 Codex 正在让技术圈意识到「AI 能工作」.

这种认知差距会造成什么后果?

- 技术圈的人在用 Agent 把工作效率提升 10 倍
- 不懂技术的人在担心 AI 会抢饭碗
- 而真正的问题可能不是「抢饭碗」,而是「会不会用 Agent」

就像文章说的:「Once a computer can use computers, you're off to the races.」

一旦电脑能使用电脑,比赛就开始了.
