发现一个有意思的功能:`claude --worktree`

Claude Code 创作者 Boris Cherny 刚刚发的.

## 谁能用?
用 Claude Code 写代码的,想同时跑多个 Agent 干活.

## 怎么用
```bash

# 在独立 worktree 里启动 Claude
claude --worktree

# 命名你的 worktree
claude --worktree --name my-feature

# 加上 tmux 分屏
claude --worktree --tmux
```

每个 Agent 有自己的 worktree,互不干扰.

## 有啥特别的?
**并行 3-5 个 Agent**:Boris 说这是团队最大的生产力提升.

- Agent A 写 API
- Agent B 写前端
- Agent C 写测试

同时跑,代码不会互相覆盖.

**--tmux 集成**:每个会话在独立 tmux 窗口,管理更方便.

## 跟手动开 worktree 比?
手动开 worktree 也能做到,但 `--worktree` 自动帮你:
- 创建 worktree, 隔离环境, 管理会话

一条命令搞定.
