心理健康 AI 应用大多是闭源的,数据要上传到别人的服务器.

**OpenGnothia** 不一样:开源,本地存储,BYOK.你的隐私你自己做主.

## 名字来源
"Gnôthi Seautón"(Γνῶθι Σεαυτόν)--德尔斐神谕上的铭文:**认识你自己**.

## 核心功能

### AI 治疗会话
- 实时流式响应, 会话计时器, 自动总结:识别主题,防御机制,洞察,家庭作业建议

### 日记
- 每日记录, 情绪标签, 可选 AI 分析,发现模式

### 梦境分析
- 记录梦境, AI 辅助解读, 可搜索的梦境日记

### 情绪追踪
- 每日心情,能量,睡眠质量, 日历视图, 可视化情绪趋势

### 呼吸练习
- 多种呼吸模式
- 管理压力和焦虑

### 治疗计划
- 结构化自我探索
- 渐进式练习和反思

## 6 种疗法流派
精神分析, 创始人=弗洛伊德, 焦点=潜意识,过去经历,关系模式
CBT, 创始人=Aaron Beck, 焦点=识别和改变思维模式
意义疗法, 创始人=Viktor Frankl, 焦点=在苦难中寻找意义
ACT, 创始人=Steven Hayes, 焦点=心理灵活性,价值观驱动
图式治疗, 创始人=Jeffrey Young, 焦点=识别早期适应不良图式
斯多葛主义, 创始人=马可·奥勒留, 焦点=内心平静,美德,理性

每种流派塑造 AI 如何倾听,回应和引导你的会话.

## 隐私设计
- **本地存储**:SQLite,不上云, **无遥测**:不收集任何数据, **BYOK**:用自己的 API Key, **生物识别锁**:Touch ID(macOS), **密码保护**

唯一的外部通信是你选择的 AI 提供商,用你自己的 Key.

## 支持的 AI 提供商
- **Anthropic Claude**:Opus,Sonnet,Haiku(支持扩展思考)
- **OpenAI**:GPT-5 系列,GPT-4.1,GPT-4o
- **自定义端点**:任何 OpenAI 兼容 API

## 技术栈
- **前端**:React 19 + TypeScript + TailwindCSS 4
- **桌面运行时**:Tauri 2, **后端**:Rust, **数据库**:SQLite

## 安装

### 下载
从 [GitHub Releases](https://github.com/Lepuz-coder/opengnothia/releases) 下载.

### macOS 提示
如果看到「应用已损坏」警告:

```bash
xattr -cr /Applications/OpenGnothia.app
```

### 从源码构建
git clone https://github.com/Lepuz-coder/opengnothia.git
cd opengnothia
pnpm install
pnpm tauri dev

## 首次启动
1. 选择 AI 提供商
2. 输入 API Key(本地存储)
3. 选择模型
4. 选择疗法流派
5. 设置名字和偏好

## 我的看法
心理健康 AI 是敏感领域.闭源 + 云端存储的组合,对很多人来说是障碍.

OpenGnothia 的设计原则是对的:
- **开源** = 透明,可审计, **本地** = 隐私保护, **BYOK** = 控制权

作者是团队负责人,业余时间用 Claude Code 做的.用 Opus + 扩展思考模式,每天 15-20 分钟,几周后 AI 能识别出你的思维模式--回避行为,认知扭曲等.

这不是替代专业治疗,但作为日常自我反思工具,很有价值.
