## 🤔 Mac mini 买不起，AI 助理就不能玩了？

最近 OpenClaw 火得一塌糊涂，搞得 Mac mini 都快卖断货了。但说实话，**不是每个人都想花几千块买台电脑就为了跑个 AI 助理**。

有没有便宜到离谱的方案？还真有——Sipeed 刚开源的 **PicoClaw**，号称 10 美元硬件就能跑，内存占用不到 10MB。你没看错，是 MB 不是 GB。

## 🚀 三个让人眼前一亮的点

### 1. 极致轻量：10MB 内存，1 秒启动

PicoClaw 用 Go 语言编写，编译成单个二进制文件，**支持 RISC-V、ARM、x86 三大平台**。在一个单核处理器的小开发板上，冷启动只要 1 秒。对比 OpenClaw 需要 Node.js 运行时和几个 GB 的内存，PicoClaw 的资源占用少了 99%。

### 2. AI 自举：95% 代码由 AI 生成

这可能是最赛博朋克的部分——PicoClaw 的核心代码有 **95% 是 AI Agent 自己写的**，人类只负责监督和微调。一个 AI 设计了另一个 AI 的架构，还优化了自己的代码。套娃成精了。

### 3. 麻雀虽小五脏俱全

别看它小，该有的都有：
- ✅ 任务规划和日志记录
- ✅ 网页搜索
- ✅ 自动化工作流
- ✅ 接入 Telegram / Discord
- ✅ 支持任意 LLM 提供商（配个 API Key 就行）

## 🛠️ 三步上手

```bash
# 1. 下载对应平台的二进制文件
wget https://github.com/sipeed/picoclaw/releases/latest/download/picoclaw-linux-riscv64

# 2. 初始化工作空间
./picoclaw init

# 3. 配置 API Key，开聊
./picoclaw start
```

就这么简单。不需要 Docker，不需要 Node.js，甚至不需要 Python。一个二进制文件走天下。

## 💡 极客点评

PicoClaw 走的是和 OpenClaw 完全不同的路线。**OpenClaw 追求功能全面、生态丰富，PicoClaw 追求极致精简、到处能跑**。

**适合谁用？**
- 🔧 嵌入式开发者：在开发板上跑个智能助手监控系统
- 💸 预算党：10 美元入门 AI 助理，学生党福音
- 🏠 智能家居玩家：扔个小板子在角落当语音管家
- 🧪 极客实验：RISC-V 开发板 + AI，赛博朋克感拉满

**不适合谁？**
- 需要复杂 Skill 生态的重度用户（OpenClaw 的 ClawHub 生态暂时无法替代）
- 对延迟敏感的实时交互场景（毕竟硬件性能有限）

MIT 开源协议，代码在 [GitHub](https://github.com/sipeed/picoclaw) 上。感兴趣的可以去 Star 一下，或者直接找块吃灰的开发板试试。
