## 一句话概括
Google 搞了个叫 UCP(Universal Commerce Protocol)的开放协议,让 AI Agent 可以直接帮你在搜索结果里完成「发现商品→加购→结账→追踪物流」的全流程.上周已经在 Google AI Mode 里正式上线,Etsy 和 Wayfair 是第一批吃螃蟹的.

听起来像是「又一个 Google 协议」?别急,这次的影响可能比你想的大得多.

## 发生了什么
2026年1月,Google 联合 Shopify,Etsy,Wayfair,Target,Walmart 等电商巨头发布了 UCP 协议.2月12日,UCP 驱动的结账功能在 Google AI Mode(搜索和 Gemini App)中正式上线.

实际效果是这样的:你在 Google AI Mode 里搜「适合小户型的书架」,AI 不只是给你推荐链接,而是直接展示商品详情,旁边有个「Buy」按钮.点击后,用 Google Wallet 里存的支付方式和地址,几步就能完成购买--**全程不用离开搜索页面**.

SEO 圈子已经炸了.Glenn Gabe 在 X 上发了实测截图:「这对电商,对 Google,对 AI 搜索来说都是大事.」

## UCP 到底是什么
别被「协议」这个词吓到,UCP 的核心逻辑非常清晰:

**三个核心能力:**
- **Identity Linking**:基于 OAuth 2.0,让 AI Agent 代表用户在商家网站执行操作
- **Checkout**:创建购物会话,管理购物车,计算总价,选择支付和配送方式,完成交易
- **Order Management**:追踪物流,发起退货,更新配送状态

**两个扩展机制:**
- **Extensions**:可选功能模块,比如折扣,优惠券,会员积分
- **Services**:通信层,支持 REST API 和 MCP(Model Context Protocol)

重点来了--UCP 兼容 MCP.这意味着任何支持 MCP 的 AI Agent,理论上都能接入 UCP 生态.不只是 Google 的 Gemini,Claude,GPT,开源 Agent 都有可能.

## 为什么这次不一样
Google 搞过很多协议和标准,大多数都凉了.但 UCP 有几个不同的点:

**1. 时机对了**

AI Agent 正处于从「能聊天」到「能干活」的拐点.MCP 协议在2025年火了之后,Agent 能调用工具了,但「购物」这个最大的互联网场景还缺一个标准化的接口.UCP 刚好填上这个空白.

**2. 玩家够重**

Shopify 占全球电商 10%+ 的份额,Walmart 和 Target 是美国前三的零售商.这不是 Google 自嗨,而是电商产业链的核心玩家都下场了.

**3. 开放标准,不是封闭花园**

UCP 文档在 [ucp.dev](https://ucp.dev/latest/) 公开,任何人都可以实现.这跟当年 Google 搞 AMP(表面开放实际锁定)不一样--至少目前看,UCP 没有强制要求流量必须走 Google.

**4. 已经跑起来了**

不是画饼,Etsy 和 Wayfair 已经上线,Shopify/Target/Walmart 即将跟进.有实际用户在用了.

## 对开发者意味着什么
如果你在做 AI Agent 相关的开发,UCP 值得关注:

- **电商 Agent 有标准可循了**:不用再给每个电商平台写自定义爬虫和 API 适配
- **MCP + UCP 组合拳**:MCP 管工具调用,UCP 管商业交易,Agent 的「能力栈」更完整了
- **新的创业机会**:帮中小商家接入 UCP,做 UCP 中间件,做跨协议的 Agent 编排

不过也别太乐观.目前 UCP 只在美国市场上线,覆盖的商家还很少.而且 Google 在中间的角色--支付走 Google Wallet,发现走 Google Search--意味着 Google 依然是流量和交易的核心节点.

## 电商格局的深层影响
往大了说,UCP 代表的是一个趋势:**购物的入口正在从「网站/App」变成「AI 对话」**.

想想这意味着什么:
- **SEO 要变天**:用户不再浏览搜索结果页,而是让 AI 直接推荐和下单.传统的「排名→点击→转化」链路要被压缩成「AI推荐→Buy」
- **品牌官网的价值被稀释**:如果用户在 AI 界面就能完成全部流程,商家官网就退化成了一个仓库后台
- **中间商困局**:导购网站,比价平台,返利网站--这些依赖「用户主动浏览」的商业模式会受到冲击

当然,这一切还在很早期.但方向已经很清楚了.

## 我的判断
UCP 大概率会成为 AI 电商的事实标准--不是因为它技术有多先进,而是因为 Google 手里有搜索流量和支付基础设施,再加上 Shopify+Walmart+Target 的背书,其他玩家很难另起炉灶.

但风险也很明显:
- Google 会不会在 UCP 里塞广告优先级?(大概率会)
- 隐私问题--AI Agent 代你购物,意味着更多消费行为数据被平台掌握
- 商家的议价能力进一步被削弱--当 AI 成为购物入口,「AI 推荐你」就变成了新的「搜索排名」

总结:UCP 是 AI Agent 时代的 HTTP--一个基础协议,会改变电商的底层游戏规则.现在还早,但值得所有做 AI 和电商的人密切关注.
