发现一个有意思的东西:MBC

Laravel 后端也能跑 AI Agent 了.

## 谁能用?
PHP/Laravel 开发者,想在服务端跑 AI 自动化任务.

## 怎么用
安装:

```bash
composer require undergrace/laravel-mbc
php artisan vendor:publish --tag=mbc-config
php artisan migrate
```

配置 .env:

MBC_PROVIDER=anthropic
ANTHROPIC_API_KEY=sk-ant-...

写代码:

```php
$session = Mbc::session('my-agent')
 ->systemPrompt('你是数据助手')
 ->tools([ListItemsTool::class, CreateReportTool::class])
 ->context(['user_id' => 1])
 ->start('分析销售数据并生成报告');

## 有啥特别的?
**多 Provider 支持**:
- Anthropic (Claude)
- OpenAI (GPT-4o, o1, o3)
- OpenRouter (200+ 模型)

**多 Agent 模式**:
- Pipeline:流水线, Orchestrator:编排, Shared Memory:共享内存

**工具用 PHP Attributes 定义**:

#[Tool(name: 'analyze_data', description: '分析数据')]
class AnalyzeDataTool extends BaseTool
{
 #[ToolParam(name: 'table', required: true)]
 public function execute(array $input): mixed
 return $this->dataRepo->analyze($input['table']);
 }
