以前 context-switch 一天三个功能.现在午休前写完三个描述,下午看 PR 就行.

Paragent 的思路挺简单:你用英语描述要什么,它开一个新分支,规划,写代码,跑测试,最后给你开个 PR.

重点是--可以并行跑 10 个.

每个 Agent 独立分支,不会打架.写完的 PR 就是标准 GitHub 流程,该 review review,该 merge merge.

几个我觉得不错的点:
- BYO API key:用你自己的 OpenAI/Anthropic/Gemini key,它不存代码也不转卖 token
- 不碰 main:GitHub App 只有读和开 PR 的权限,永远不直接改主分支
- 不限运行次数:只收席位费,API 费用你自己控制

免费版 2 个并发,Pro 版 $14/月 10 个并发,Agency 版 $99/月不限.

https://paragent.app/
