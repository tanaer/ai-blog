发现一个有意思的东西:zclaw

888 KB 的 AI 助手,跑在 ESP32 上.

## 谁能用?
喜欢折腾硬件的,想要离线 AI 助手的,IoT 开发者.

## 怎么用
一键安装:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/tnm/zclaw/main/scripts/bootstrap.sh)
```

然后:
1. 烧录固件到 ESP32
2. 配置 WiFi + LLM 凭证
3. 通过 Telegram 或 Web 聊天

## 功能
- **Telegram / Web 聊天**
- **时区感知的计划任务**(每天,周期,一次性), **GPIO 读写控制**(带安全保护), **持久内存**(重启不丢), **多 Provider**:Anthropic,OpenAI,OpenRouter

## 硬件
支持 ESP32-C3,ESP32-S3,ESP32-C6.

推荐入门板:Seeed XIAO ESP32-C3(几块钱)

## 跟云端助手比?
- 离线运行(WiFi 只是调用 API), 你的硬件,你的数据, 可以控制 GPIO(智能家居), 888 KB,你能读完源码
