## 发现一个有意思的工具
PRScope 用 AI 自动审查 GitHub PR,30 秒配好,一分钟出报告.关键是 -- 不用注册,你的 API key,你的模型.

## 谁能用?解决什么问题?
**场景**:团队 PR 多,人工 review 来不及,问题溜进代码库.

**PRScope 干的事**:
- 读 PR diff,理解上下文, 用你选的 LLM 分析, 在 PR 下发一条结构化评论

**支持的模型**:
- OpenAI:GPT-4.1,o3,o4-mini, Anthropic:Claude Sonnet 4,3.7,3.5, Ollama:Llama 3.3,Qwen 2.5,DeepSeek(本地跑), 任何 OpenAI 兼容端点

**四种审查模式**:

- balanced: Bug,安全,性能,代码质量, security: 注入攻击,认证缺陷,数据泄露, performance: 内存泄漏,算法问题, strict: 全都要,风格命名文档都不放过

## 相比现有方案好在哪?
- **无锁定**:没有账号系统,API key 存本地 `.env`
- **数据不离手**:diff 只发给你选的 LLM,用 Ollama 的话全在本地
- **纯 Markdown**:没有图片,没有外链,没有 JavaScript,任何 GitHub 套餐都能用
- **风险评级**:低/中/高,一眼看出严重程度

还会"夸人" -- 报告里有 Praise 部分,告诉你哪里写得好.

## 怎么用
```bash

# 初始化(有向导)
npx prscope init

# 生成两个文件:prscope.config.json + .github/workflows/prscope.yml
```

把 API key 加到 GitHub Secrets,提交代码.下次有人开 PR,PRScope 自动审查.
