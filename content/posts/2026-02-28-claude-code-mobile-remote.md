## 手机当遥控器，电脑干活

Anthropic 刚给 Claude Code 加了个新功能。

用手机就能控制电脑上的代码会话。

不是把计算搬到云端。

是你的电脑在跑，手机只是遥控器。

打开 Claude 移动应用，就能看到电脑上的终端。

在哪都能接着干活。

地铁上、咖啡馆、甚至躺着。

代码不会停。

## 安全吗？人家想过了

很多人第一反应：会不会不安全？

Anthropic 说不会暴露本地系统。

没有入站连接。

你的电脑不用对外开门。

所有同步都走官方通道。

就像你用手机看家里的监控。

监控在你家，你只是远程看。

不过这功能现在只给 Claude Max 用户。

订阅费一个月 100-200 美元。

贵是贵了点，但重度用户可能觉得值。

## 25 亿年化，说明有人买单

Claude Code 这个产品挺能打。

报道说年化运行率已经 25 亿美元。

什么概念？

就是用户愿意付钱。

这次加移动控制，明显是想让用户更离不开。

以前写代码得坐在电脑前。

现在随时能接上。

改个 bug、推个代码，手机点几下就行。

不需要回到办公桌。

对独立开发者和小团队尤其友好。

## 营销人也要关注这件事

表面上看是程序员工具。

但背后的趋势值得所有人注意。

移动端 AI 代理来了。

时间和位置的限制在瓦解。

营销人想想看。

以前做内容、搞活动，得在电脑前。

以后可能手机上就能跑完整流程。

AI 在电脑上干活，你在外面盯进度。

白天开会，晚上手机上审稿。

创作和迭代的周期会被压缩。

这不是科幻，已经开始发生了。

## 该怎么做？

如果你是程序员，试试 Claude Max。

体验一下移动控制的便利。

如果你做营销、做产品，关注这个趋势。

AI 工具正在打破场景限制。

未来可能所有创作工具都会移动化。

早点熟悉，早点受益。

别等大家都用熟了你才反应过来。

---

**一句话总结**：Claude Code 移动端远程控制，让写代码不再被电脑绑死。AI 工具移动化的大趋势，值得关注。
