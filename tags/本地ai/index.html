重磅:ggml.ai 创始团队加入 Hugging Face,llama.cpp 迎来新篇章

本地 AI 领域迎来重大消息:ggml.ai 创始团队正式加入 Hugging Face. Georgi Gerganov 和他的团队–llama.cpp 的创造者–现在成为 Hugging Face 的全职员工.这不是收购,而是一次战略合作,目标是"确保未来 AI 真正开放". 关键信息 不会改变: ggml-org 项目保持开源和社区驱动 Georgi 和团队继续 100% 时间维护 ggml/llama.cpp 社区保持完全自治,技术和架构决策不变 项目继续 100% 开源 会改变: Hugging Face 提供长期可持续资源, 更快的模型支持(你最喜欢的量化模型会更快上线), 更好的 transformers 库集成, 改善用户体验和包装 为什么是 Hugging Face? 过去几年,Hugging Face 一直是 ggml 最坚定的支持者: 贡献了多个 ggml/llama.cpp 核心功能 构建了 solid 的推理服务器和精美 UI 为 llama.cpp 引入多模态支持 将 llama.cpp 集成到 Hugging Face Inference Endpoints 改进了 GGUF 格式与 HF 平台的兼容性 实现了多种模型架构 帮助日常维护和 PR 审查 Georgi 说:“双方合作一直很顺畅高效,社区也从中受益.正式化这个合作,未来会更强.” ...
