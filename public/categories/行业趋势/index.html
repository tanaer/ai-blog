Atlantic 深度:AI Agent 时代已经来了,你准备好了吗?

The Atlantic 刚刚发了一篇深度报道,标题很直接:AI Agents Are Taking America by Storm. 不是「将要」,是「正在」. 两个平行宇宙 文章描述了一个有趣的现象:美国人正在生活在两个平行的 AI 宇宙中. 宇宙一:大多数人.对他们来说,AI = ChatGPT + Google AI Overview + 社交媒体上的 AI 垃圾内容. 宇宙二:技术爱好者.他们正在被那些能连续工作数小时的机器人「激进化」–把几个月的工作压缩成几周,把几周压缩成一下午. 这两个宇宙之间的差距正在迅速扩大. 数据:AI 写了多少代码? 微软 CEO Satya Nadella 说:30% 的微软代码现在是 AI 写的.微软 CTO 预计,到本十年末,这个数字会达到 95%. Anthropic 内部的比例更高:最高 90% 的代码是 AI 生成的. Salvatore Sanfilippo(Redis 创始人)最近写了一篇爆款文章:「对大多数项目来说,自己写代码已经不再合理了」. 后 ChatGPT 时代 文章的核心论点是:我们正在进入「后 ChatGPT 时代」. 传统的聊天机器人(ChatGPT,Gemini)已经很强了: 记忆功能:能引用之前的对话, 推理能力:能产出更复杂的回答, 多模态:能处理书级别长度的文件,图片,视频,音频 但跟 AI Agent 相比,这些都是小儿科. Agent 的核心区别:不是帮你回答问题,是帮你干活. 实际案例 文章列举了一些真实的使用场景: 学术界:用 Claude Code 自主生成论文 生物研究:用 Agent 做实验设计和数据分析, 新闻业:用 Agent 从零开始写数据驱动的报道, 创业者:两个人在一小时内做出 Monday.com(市值数十亿美元的公司)的竞品原型 作者自己的体验: ...
