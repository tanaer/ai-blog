Expo-OpenClaw-chat:在 React Native 应用里内置 AI 聊天,边聊边改代码

想象一下:用户在用你的产品时遇到问题,直接在应用里打开聊天,AI 不仅回答问题,还能实时修改应用代码解决问题. 这就是 Expo-OpenClaw-chat – 一个为 Expo/React Native 应用打造的 OpenClaw 聊天 SDK. 核心特性 最小化集成 import { createChat } from "expo-openclaw-chat"; const chat = createChat({ gatewayUrl: "wss://your-gateway.example.com", token: "your-auth-token", }); // 在应用任何地方打开聊天 chat.open(); 实时修改应用 AI 通过聊天理解用户需求, 直接修改应用代码(类似 vibecoding), 用户无需离开应用即可获得修复 完整聊天功能 图片附件(支持 expo-image-picker), Markdown 渲染(支持 react-native-marked), 消息历史, 打字指示器 安全认证 自动 Ed25519 密钥对生成 支持 expo-secure-store(iOS Keychain/Secure Enclave) 设备身份持久化(支持 react-native-mmkv) 快速开始 npm install expo-openclaw-chat # 可选依赖 npm install expo-image-picker # 图片附件 npm install react-native-marked # Markdown 渲染 npm install expo-secure-store # 安全密钥存储 npm install react-native-mmkv # 持久化存储 import { MMKV } from "react-native-mmkv"; import { setStorage } from "expo-openclaw-chat"; // 配置持久化存储 const storage = new MMKV({ id: "my-app" }); setStorage(storage); // 创建聊天实例 gatewayUrl: "wss://gateway.example.com", token: "your-token", title: "AI 助手", placeholder: "描述你遇到的问题...", showImagePicker: true, onOpen: () => console.log("聊天已打开"), onClose: () => console.log("聊天已关闭"), // 包装应用 function App() { return ( <chat.ChatProvider> <YourApp /> </chat.ChatProvider> ); } // 从任何地方打开聊天 ## 配置选项 ```typescript interface ChatConfig { // 必需 gatewayUrl: string; // WebSocket URL (wss:// or ws://) // 认证(三选一) token?: string; // 简单 auth token password?: string; // 密码认证 deviceToken?: string; // 配对设备 token // 可选 sessionKey?: string; // 会话 key(自动生成) title?: string; // 弹窗标题 placeholder?: string; // 输入框占位符 showImagePicker?: boolean; // 显示图片选择按钮 clientId?: string; // 客户端 ID // 回调 onOpen?: () => void;
