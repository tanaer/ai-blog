Spackle:Mac 全局 AI 重写快捷键,选中文字按一下就改好

在 Mail 里写邮件,想改一下措辞.切到 ChatGPT,复制,粘贴,等回复,复制,切回来,粘贴……累不累? Spackle 解决了这个问题:选中文字,按快捷键,原地替换. 一句话介绍 Spackle 是一个 macOS 菜单栏应用,让你在任意 App 里用 AI 重写选中的文字–不用切窗口,不用复制粘贴. 怎么用? 选中任意文字(Mail,Notes,Slack,浏览器,编辑器…) 按快捷键 ⌘⌥⌃I(可自定义) AI 重写后的文字直接替换原文 就这么简单.没有聊天窗口,没有来回切换. 支持什么模型? Bring Your Own API Key: OpenAI, Anthropic, OpenRouter, 任何 OpenAI 兼容端点 API Key 存在 macOS Keychain 里,不上传到任何地方. 实际应用场景 写邮件 草稿写完了觉得语气太生硬?选中,按快捷键,改成更友好的版本. Slack 回复 同事发了个长问题,你想简洁回复.选中草稿,让 AI 精简一下. 写代码注释 写了一堆啰嗦的注释?选中,一键改成清晰简洁版. 任何文本输入框 浏览器表单,Notion,VS Code……只要能选中文字,就能用. 设置 # 4. 设置快捷键 额外功能:触发器模式 如果你不喜欢快捷键,还可以用文字触发器: 输入特定前缀触发重写 或者用语音命令触发 隐私 只有当前选中的文字会被发送到你配置的 AI Provider.其他什么都不收集. 下载 官网:https://aisatsu.co/spackle/ GitHub:https://github.com/tsortwehttam/spackle 我的看法 这类「隐形 AI 工具」是趋势–不抢占你的注意力,而是在你需要的时候静默出现. ...
