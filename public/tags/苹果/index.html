Ferret-UI Lite：苹果的 3B 参数端侧 GUI Agent，跑在手机上

📱 发现苹果发了篇有意思的论文：Ferret-UI Lite 3B 参数的 GUI Agent，跑在设备端，跨手机/Web/桌面三端。 能干什么？ 看懂屏幕上的 UI 元素 定位按钮、图标、文本位置 自动导航 App 界面 执行点击、滑动等操作 基准测试表现： GUI Grounding（定位）： ScreenSpot-V2：91.6% ScreenSpot-Pro：53.3% OSWorld-G：61.2% GUI Navigation（导航）： AndroidWorld：28.0% OSWorld：19.8% 怎么做到的？ 混合数据训练 真实 GUI 数据 + 合成数据 覆盖多平台、多分辨率 推理时增强 Chain-of-thought 推理 视觉工具调用 强化学习 设计奖励信号 针对小模型优化 跟之前什么关系？ Ferret-UI 2 是通用 UI 理解模型（支持 iPhone/Android/iPad/Web/AppleTV） Ferret-UI Lite 是轻量版，专注端侧执行，3B 参数能在手机上跑。 为什么重要？ 大多数 GUI Agent 要上云，延迟高、隐私问题大。苹果的路子是：小模型 + 端侧推理，用 RL 和 CoT 补足能力。 论文：machinelearning.apple.com/research/ferret-ui arXiv：arxiv.org/abs/2509.26539 ⚡ GUI Agent 的终局可能是：3B 参数 + 手机本地运行，不传云端。 ...
