没人打招呼,三大AI巨头却在同一天掀桌子

说实话,有点离谱. 没人打招呼,没人协调,但Anthropic、OpenAI、DeepSeek偏偏选在2026年2月同一天掀桌子. 这不是什么发布会撞车,这是一场没有硝烟的模型战争. 发生了什么? 2月5号,Anthropic丢出Claude Opus 4.6,主打一个Agent Teams——不再是单个AI干活,而是多个Agent并行协作.写代码、做分析、搞研究,原来两小时的活现在二十分钟搞定.十二天后,Sonnet 4.6跟上,直接把Opus级别的性能塞进中端价位. 同一天,OpenAI掏出GPT-5.3-Codex.最骚的是这玩意儿参与了自身的开发——调试、部署、诊断测试结果,AI写AI,递归进化开始了. 然后DeepSeek V4的预发布信号也出来了:上下文窗口从128K飙到100万token,知识库更新到2025年5月. Nomura的分析师说,正式版随时可能落地. 有意思的点 其实这次混战暴露了一个趋势:大家都不再比谁更会聊天了,比的是谁能真正干活. Claude的Excel和PowerPoint集成,Opus的Agent Teams,都在冲着"知识工作者"去.GPT-5.3-Codex死磕编程,DeepSeek继续打性价比和开源牌. Benchmark已经不重要了,重要的是谁能当你的同事. 该怎么选? 说实话,日常用的话差距没你想的大.真正拉开距离的是长任务、复杂编排、Agent自主运行这些场景. 知识工作、办公自动化 → Claude Opus/Sonnet 4.6 纯编程 → GPT-5.3-Codex 要省钱或数据不能出墙 → DeepSeek 这场仗才刚开始,谁的优势都是暂时的. 但有一点可以确定:2026年2月,是AI从"会说话"变成"会干活"的分水岭. 🔗 完整分析

GPT-5.2花12小时推导出新物理公式,AI真的开始做原创科学了?

上周末,OpenAI放了一个大招:GPT-5.2 Pro独立推导出了一个全新的理论物理公式,还花了12个小时生成了完整的形式化证明. 这不是做题.不是刷竞赛.是真正的,之前没有人发现过的物理结果. 到底发现了什么? 先说背景.在粒子物理里,有一类叫做"胶子散射振幅"的东西–简单说就是计算胶子(强力的载体粒子)碰撞概率的数学公式. 1986年,Parke和Taylor给出了一个优美的公式,描述了"最大螺旋度违反"(MHV)振幅.这个公式极其简洁,是粒子物理里的经典之作. 但有一类特殊情况–只有一个负螺旋度胶子,其余全是正螺旋度的情况–几十年来教科书都写着:这类振幅为零. GPT-5.2说:不对,它们不为零. 具体来说,在一个叫"半共线区域"(half-collinear regime)的特殊动量空间里,这些振幅其实是非零的.GPT不仅发现了这个漏洞,还给出了一个对任意n都成立的闭合公式,并用Berends-Giele递推关系和soft limit验证了正确性. 12小时的"思考"过程 论文的作者阵容相当豪华:普林斯顿高等研究院,范德堡大学,剑桥大学,哈佛大学,加上OpenAI. 流程是这样的: 人类物理学家手动计算了n=1到n=6的具体情况,得到了极其复杂的表达式 GPT-5.2 Pro拿到这些具体情况后,化简了表达式,发现了隐藏的规律,猜测出了一个对所有n都成立的通用公式 一个内部加强版GPT-5.2花了大约12小时的推理时间,独立推导出了同一个公式,并生成了完整的形式化证明 人类物理学家用标准方法验证了证明的正确性 注意第2步和第3步–AI不是在做"搜索+拼接",而是在做真正的数学推理:从有限个具体案例中归纳出通用规律,然后证明它. HN上吵翻了 Hacker News的讨论非常精彩,观点分几派: 乐观派认为这是里程碑–AI已经能在有验证测试的问题上做出真正的贡献.散射振幅的计算复杂度超指数增长,人类算到n=6就已经极其痛苦了,但AI能从中发现规律并推广到任意n. 谨慎派指出关键细节:问题的specification是人类给的.物理学家已经有了"这类振幅可能不为零"的直觉,已经算出了具体案例,然后才把"找规律+证明"的任务交给AI.AI做的是"最后一公里"–虽然这最后一公里人类确实没跑出来. 怀疑派则更尖锐:这本质上就是暴力搜索+统计偏差.把大量计算资源砸到一个有明确验证标准的问题上,偶尔找到一个解就宣称"智能"–这和真正的科学发现是一回事吗? 我觉得最精彩的一条评论是: “现代LLM在有验证测试集的问题上极其强大且不知疲倦.但这个结论不能推广到开放式问题上.发现问题的specification往往才是真正的战场.” 我的看法 这件事的重要性在于程度的变化,而不是性质的变化. AI辅助科研不是新鲜事.AlphaFold预测蛋白质结构,DeepMind的Aletheia解数学难题,都是先例.但GPT-5.2的特殊之处在于:它不是一个专门为某个任务训练的系统,而是一个通用语言模型,在给定足够的"思考时间"后,展现出了数学推理能力. 12小时的推理时间也很有意思.这说明现在的scaling不只是在模型参数上,也在推理时间(inference-time compute)上.给模型更多时间去"想",它真的能想出更深刻的东西.这和人类解难题的过程其实很像–你也不是看一眼就能证明定理的. 但我不会把这叫做"AI做科学".至少现在还不是.真正的科学发现需要的是问出正确的问题–为什么去看half-collinear regime?为什么怀疑教科书上"振幅为零"的结论?这些直觉来自人类物理学家.AI做的是在人类指定的方向上,执行了人类做不到的复杂推理. 这已经很了不起了.但"AI科学家"和"极其强大的AI科研工具"之间,还有一道关键的鸿沟. 真正的信号 抛开PR和争议,这篇论文传递的真实信号是: 推理时间scaling有用–12小时的"思考"换来了一个形式化证明,这说明inference compute的投入有明确回报 通用模型能做专业推理–不需要专门训练一个"物理AI",通用LLM在给定足够上下文后就能做非平凡的数学物理推理 人机协作的范式正在成型–人类提供直觉和方向,AI执行复杂的推导和搜索,两者互补 这个范式如果成熟,影响将远超粒子物理.材料科学,药物设计,密码学–任何有明确验证标准的理论问题,都可能被这种方式加速. 论文团队也提到,同样的方法已经被用于从胶子推广到引力子的振幅计算,更多结果即将发表. 物理学教科书可能要改了.而改动的起因,是一个语言模型花了12个小时"想"出来的. 不管你怎么定义"智能",这都值得认真对待.
