AI开上火星了!NASA用Claude让毅力号自己找路跑了456米

地球到火星,一个信号要等25分钟 你在地球上遥控一辆火星车,每发一个指令要等25分钟才能收到回应.这就是NASA工程师每天面对的现实. 传统做法是:工程师看卫星图 → 手动规划路线 → 设置不超过100米的航点 → 通过深空网络发送给轨道器 → 再转发给火星车.一套流程下来,毅力号一天能走的距离非常有限. 如果火星车能自己看路,自己决定怎么走呢? 核心亮点:AI当了火星司机 1️⃣ Claude上火星了(没开玩笑) NASA喷气推进实验室(JPL)基于Anthropic的Claude AI开发了路径规划系统.AI分析火星侦察轨道器的HiRISE高分辨率相机拍的卫星图和数字高程模型,自动识别沙坑,巨石群,基岩等障碍,生成一系列安全航点. 2️⃣ 两天跑了456米,零人工操控 2025年12月,毅力号在杰泽罗陨石坑边缘完成了两次AI规划的自主行驶,总计456米.AI规划的路线(洋红色)和实际行驶路线(橙色)高度吻合,说明AI对火星地形的理解相当靠谱. 3️⃣ 地球上先"模拟考"再上路 JPL有一辆毅力号的"双胞胎"–VSTB(车辆系统测试台),放在JPL的火星模拟场地里.AI规划的路线会先在地球上跑一遍,确认没问题再发到火星.稳得一批. ️ 这套系统怎么工作的? 轨 关 J # 信 有 > - 道 ↓ 地 深 车 键 P # * 号 意 * 卫 球 空 载 突 L 这 延 思 适 N * [ 星 模 网 自 破 * 空 极 件 迟 的 用 A 想 N 拍 拟 络 动 在 感 间 客 事 是 是 场 S 深 A 照 车 导 于 知 机 点 的 星 N 景 A 入 S 验 → 航 A * 器 评 意 际 A 延 局 了 A → 证 接 I * 人 义 探 S 伸 长 解 轨 管 承 : 专 远 索 A * J ?
