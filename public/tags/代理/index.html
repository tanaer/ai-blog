InferShield：LLM 安全代理，一行代码保护你的 API 调用

发现一个有意思的东西：InferShield 🛡️ 开源的 LLM 安全代理，保护你的 API 调用。 谁能用？ 用 OpenAI/Anthropic/Google API 的应用，担心： Prompt 注入攻击 数据泄露 合规要求（SOC 2、HIPAA、GDPR） 怎么用 零代码改动，只换 API endpoint： # 之前 client = OpenAI(base_url="https://api.openai.com/v1") # 之后 client = OpenAI(base_url="http://localhost:8000/v1") Docker 一键启动： docker run -p 8000:8000 \ -e OPENAI_API_KEY=sk-your-key \ infershield/proxy 检测什么 Prompt 注入：覆盖系统指令的尝试 数据泄露：提取敏感数据的请求 越狱攻击：编码、混淆等规避技术 SQL 注入：数据库攻击模式 PII 泄露：个人身份信息 特点 自托管：数据不离开你的基础设施 多提供商：OpenAI、Anthropic、Google、本地模型 风险评分：每个请求 0-100 分 完整审计：所有请求都有日志 链接 GitHub: https://github.com/infershield/infershield 企业级 LLM 安全，开源免费 🔥
