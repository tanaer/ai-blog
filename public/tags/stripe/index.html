Stripe Minions:每周 1300+ PR 全由 AI 写代码,人类只管 Review

Stripe 刚发了第二篇 Minions 博客,披露了一个数字:每周 1300+ 个 AI 写的 PR 被合并. 不是 AI 辅助,是 AI 全程写代码,跑测试,修 bug,提 PR,人类只做最后的 Code Review. Minions 是什么? Stripe 的内部 AI 编码代理系统,特点: 完全无人值守:给定任务后,从写代码到提 PR 全程自主, 运行在 devbox:和人类工程师用一样的云开发环境, 安全隔离:每个 devbox 独立,不接触敏感系统或客户数据, 可并行:同时跑多个 minion,像管理多个团队成员 能做什么? 修复 flaky 测试: 自动定位问题,修复,验证 实现功能: 根据需求文档写完整功能代码, 代码迁移: 批量重构,API 升级, 小型改进: 性能优化,代码清理 限制:大方向决策,系统架构设计,安全风险评估仍需人类. 工作流程 任 # # # -
