OpenAI 发布 EVMbench:AI 能不能帮你审计智能合约漏洞?

智能合约里锁着 1000 亿美元.AI 能不能帮你找漏洞?OpenAI 刚刚给了答案. EVMbench 是什么? OpenAI 联合 Paradigm 发布了 EVMbench,一个专门评估 AI Agent 智能合约安全能力的基准测试. 核心数据: 120 个精心挑选的高危漏洞 来自 40 次真实审计(主要是 Code4rena 开源竞赛) 包含 Tempo 区块链的支付场景漏洞 三种测试模式 Detect, 任务=审计合约,找出漏洞, 评分标准=召回率 + 审计奖励 Patch, 任务=修复漏洞,保持功能, 评分标准=自动化测试 + exploit 检查 Exploit, 任务=执行攻击, draining 资金, 评分标准=链上验证交易重放 最有意思的是 Exploit 模式–Agent 需要在沙盒区块链环境中端到端执行资金盗取攻击. 测试结果:GPT-5.3-Codex 达到 72.2% GPT-5.3-Codex: 72.2% GPT-5: 31.9% 6 个月,性能翻了一倍多. 但 Detect 和 Patch 模式仍然挑战很大: Detect:Agent 找到一个问题就停了,不够全面 Patch:修复漏洞的同时保持完整功能很难 为什么 Exploit 分数最高? OpenAI 的分析很有意思: “在 exploit 设置中,目标非常明确:持续迭代直到资金被转走.” ...
