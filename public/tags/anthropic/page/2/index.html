全球AI大佬齐聚新德里:印度AI峰会背后的野心与博弈

硅谷半壁江山飞到了新德里 今天(2月16日),印度AI Impact Summit 2026在新德里的Bharat Mandapam正式开幕,持续到2月20日. 出席名单堪称AI界的"复仇者联盟": Sam Altman(OpenAI CEO), Sundar Pichai(Alphabet/Google CEO), Dario Amodei(Anthropic CEO) Demis Hassabis(Google DeepMind CEO) Bill Gates(微软联合创始人), Yann LeCun(Meta首席AI科学家), Yoshua Bengio(图灵奖得主), Mukesh Ambani(印度首富,Reliance掌门人) 唯一缺席的重量级选手是Jensen Huang.NVIDIA CEO原本确认参加,但2月14日突然以"不可预见的情况"为由退出.至于具体原因,NVIDIA没多解释. 全球南方的第一次 这是AI峰会系列第一次来到"全球南方".之前的举办地分别是英国(Bletchley Park, 2023),韩国(首尔, 2024)和法国(巴黎, 2025).选址印度本身就是一个信号–AI治理的话语权不能只属于发达国家. 但说实话,这次峰会的商业味道远浓于治理讨论. 印度想要什么? 三个字:基础设施,用户,人才. 基础设施方面,印度正在疯狂招商.去年12月,亚马逊,微软和英特尔已经承诺在印度建设AI数据中心和芯片制造设施.印度政府批准了180亿美元的半导体项目.这次峰会预计会有更多投资协议落地. 用户方面,数据很说明问题.Altman透露,印度是ChatGPT全球第二大市场(仅次于美国),而且拥有全球最多的学生用户群.Perplexity等竞品也在印度免费推广,抢用户的同时也在抢训练数据. 人才方面,印度的工程师大军是全球科技公司的"隐形引擎".ANSR的CEO Lalit Ahuja说得直白:“所有人都来了,因为他们意识到这个市场不能忽视.” 莫迪的算盘 莫迪政府的战略很清晰:用市场换技术,用政策换投资. 这和中美贸易框架的重置是同步的.就在两周前,美印刚推进了临时贸易协议框架.科技公司需要在中国之外找到下一个十亿用户市场,而印度是唯一够格的选手. 苹果已经在印度扩大制造,风投疯狂押注印度创业公司,IPO市场火爆.AI峰会只是这盘大棋的最新一步. 但印度AI的短板也很明显 没有本土大模型.目前印度没有一个能跟ChatGPT,Claude,Gemini抗衡的国产AI产品.这意味着印度在AI应用层繁荣的同时,底层能力完全依赖美国公司. 算力依赖进口.180亿美元的芯片计划听着豪气,但从批准到量产至少还要3-5年.短期内印度的AI算力还是得靠NVIDIA的GPU. 监管框架模糊.印度至今没有专门的AI立法.莫迪政府一直在"促发展"和"管风险"之间摇摆,这次峰会能不能拿出实质性的治理方案还是问号. 黄仁勋为什么没来? 这可能是整个峰会最有意思的悬念. NVIDIA没给具体理由,只说了"unforeseen circumstances".考虑到当前中美科技博弈的大背景,以及NVIDIA在对华芯片出口上的敏感地位,临时退出可能有更复杂的考量. 当然也可能真的只是行程冲突.但在这种级别的峰会上放鸽子,本身就是一种信号. 我的判断 这次峰会最大的看点不在于技术突破–别指望Altman会在台上发布GPT-6. 真正值得关注的是投资协议和数据治理立场. 印度如果能在AI数据主权上提出有说服力的框架(而不只是喊口号),就能真正影响全球AI治理的走向.毕竟,14亿人的数据归谁用,怎么用,这个问题的答案将直接影响下一代AI模型的能力上限. 接下来五天,我会持续关注峰会进展.有重大消息随时更新. 信息来源:CNBC,Times of India,Indian Express,Business Standard

AI开上火星了!NASA用Claude让毅力号自己找路跑了456米

地球到火星,一个信号要等25分钟 你在地球上遥控一辆火星车,每发一个指令要等25分钟才能收到回应.这就是NASA工程师每天面对的现实. 传统做法是:工程师看卫星图 → 手动规划路线 → 设置不超过100米的航点 → 通过深空网络发送给轨道器 → 再转发给火星车.一套流程下来,毅力号一天能走的距离非常有限. 如果火星车能自己看路,自己决定怎么走呢? 核心亮点:AI当了火星司机 1️⃣ Claude上火星了(没开玩笑) NASA喷气推进实验室(JPL)基于Anthropic的Claude AI开发了路径规划系统.AI分析火星侦察轨道器的HiRISE高分辨率相机拍的卫星图和数字高程模型,自动识别沙坑,巨石群,基岩等障碍,生成一系列安全航点. 2️⃣ 两天跑了456米,零人工操控 2025年12月,毅力号在杰泽罗陨石坑边缘完成了两次AI规划的自主行驶,总计456米.AI规划的路线(洋红色)和实际行驶路线(橙色)高度吻合,说明AI对火星地形的理解相当靠谱. 3️⃣ 地球上先"模拟考"再上路 JPL有一辆毅力号的"双胞胎"–VSTB(车辆系统测试台),放在JPL的火星模拟场地里.AI规划的路线会先在地球上跑一遍,确认没问题再发到火星.稳得一批. ️ 这套系统怎么工作的? 轨 关 J # 信 有 > - 道 ↓ 地 深 车 键 P # * 号 意 * 卫 球 空 载 突 L 这 延 思 适 N * [ 星 模 网 自 破 * 空 极 件 迟 的 用 A 想 N 拍 拟 络 动 在 感 间 客 事 是 是 场 S 深 A 照 车 导 于 知 机 点 的 星 N 景 A 入 S 验 → 航 A * 器 评 意 际 A 延 局 了 A → 证 接 I * 人 义 探 S 伸 长 解 轨 管 承 : 专 远 索 A * J ? 五角大楼怒了!威胁切断Anthropic合作--AI安全护栏该不该为军方让路?

当AI安全遇上军方需求,火药味十足 你可能想不到,全球最重视AI安全的公司Anthropic,正在跟全球最大的甲方–美国五角大楼,闹掰了. 事情是这样的:五角大楼要求四家AI公司(OpenAI,Google,xAI,Anthropic)开放模型给军方"所有合法用途",包括武器研发,情报收集和战场作战.其他三家不同程度地点了头,但Anthropic死活不松口. 五角大楼的态度很明确:谈了好几个月了,Anthropic再不答应,就准备"分手". 更讽刺的是,就在几天前,Anthropic的Claude模型刚刚通过与Palantir的合作,被用在了美军抓捕委内瑞拉前总统马杜罗的行动中.所以,不是完全不合作–只是Anthropic画了一条红线. 这件事为什么重要?三个关键看点 1️⃣ Anthropic的红线在哪? Anthropic表示,他们与政府的对话聚焦在使用政策层面,核心底线是两条: 完全自主武器(没有人类决策的杀人机器) 大规模国内监控 换句话说,Anthropic不是拒绝跟军方合作,而是拒绝"无条件开放".这跟他们一贯的"负责任AI"定位一致. 2️⃣ 其他公司怎么选的? 据报道,四家公司的态度分化明显: 一家已经完全同意五角大楼的条件, 两家"比Anthropic灵活",正在谈判, Anthropic是唯一硬顶的 这意味着AI行业在"军用AI"这件事上,正在走向分裂. 3️⃣ 分类网络的推进 五角大楼还在推动这些公司将AI工具部署到保密网络中,减少常规的安全限制.这不只是政策层面的事–是要把AI模型直接塞进军事机密系统. ️ 普通人怎么理解这件事? 一张图看懂利益关系: ️ 五角大楼:想要AI无限制服务于国防 → “我们是合法使用” Anthropic:坚持AI安全底线 → “合法≠没有限制”, 商业现实:五角大楼是超级大客户,拒绝意味着丢掉巨额合同, ️ 行业影响:如果Anthropic让步,“负责任AI"就成了营销口号 对你的影响: 你可能觉得这跟自己没关系,但想想–如果AI公司为了拿军方合同,逐步放弃安全限制,这些模型的商用版本还会保持严格的安全标准吗?军用AI的放松,往往会传导到民用AI. 极客点评 说实话,我对Anthropic的态度是又佩服又担心. 佩服的是,在真金白银面前坚持原则确实不容易.当你的竞争对手都在向甲方妥协的时候,你还能说"不”,这需要勇气. 担心的是,商业现实很残酷.如果Anthropic因此被排除在政府合同之外,而OpenAI和Google吃下这块蛋糕,长期来看Anthropic的资金和算力竞争力会不会受影响?“负责任AI"需要钱来支撑研究. 更深层的问题是:AI安全的边界到底该由谁来定? 是AI公司自己?政府?还是某种独立机构?目前看来,是谁有钱谁说了算. 适合关注这件事的人: 关心AI安全和伦理的从业者, 使用Claude/GPT等工具的开发者(政策变化可能影响API限制), 对AI军事化应用感兴趣的观察者 这场博弈还没结束,但不管结局如何,它已经暴露了一个根本矛盾:当"让AI更安全"和"让AI更有用"发生冲突时,钱站在哪一边?

Anthropic安全负责人辞职写诗去了!AI安全到底怎么了?

一个安全研究员的「诗意」离场 你能想象吗?一个在全球最注重AI安全的公司–Anthropic(就是做Claude那个)–负责AI安全防线的核心人物,突然宣布辞职,理由是要去学写诗? 这不是段子.2月9日,Anthropic安全防护研究团队负责人 Mrinank Sharma 在X上发了一封辞职信,原话是: “世界正处于危险之中.不仅仅是AI,不仅仅是生物武器,而是一系列相互交织的危机正在此刻展开.” 他说自己"反复看到,让价值观真正指导行动有多么困难"–包括在Anthropic内部,公司也"不断面临搁置最重要事项的压力". 然后他说要回英国,学诗歌,“让自己消失一段时间”. 三个核心看点 1️⃣ 不是第一个,也不会是最后一个 近几个月,AI大厂的安全团队正在经历一波「离职潮」.从OpenAI到Google再到Anthropic,越来越多负责"刹车"的人选择离开.问题很明显:当商业利益和安全原则冲突时,安全往往是让步的那个. 2️⃣ Anthropic的"安全人设"裂了? Anthropic一直以"我们比OpenAI更重视安全"为卖点.它甚至刚怼了OpenAI在ChatGPT里加广告.但当自家安全负责人都说"价值观很难真正落地"的时候,这个人设就很微妙了.更别提2025年Anthropic还因为训练数据版权问题赔了15亿美元. 3️⃣ AI安全不只是技术问题 Sharma的辞职信里提到的不只是AI风险,还有生物武器,社会危机等"相互交织"的问题.他之前的研究包括AI如何被用于生物恐怖主义,AI助手如何让人类"变得不像人".这些听着像科幻,但都是他在Anthropic的实际研究课题. ️ 普通人该怎么看? 不用恐慌:AI目前还远没到"毁灭人类"的程度,但"被滥用"是真实的风险 关注安全报告:Anthropic,OpenAI等公司都会发布安全评估报告,值得一读 支持监管:印度AI峰会(2月16日)即将召开,Sam Altman,Pichai,Amodei等大佬齐聚,AI治理是核心议题 保持批判:任何公司说"我们最安全",都要打个问号–连内部人都觉得价值观难以落地 极客点评 说实话,Sharma辞职去学诗这件事,表面荒诞,内核很真实. 当一个人在AI安全最前线工作了几年,最后的结论是"世界的问题太大了,我要去写诗"–这本身就是一种无声的控诉.AI行业现在的节奏是:融资→发产品→抢市场→安全?以后再说. 谁适合关注这件事? 所有使用AI产品的人.因为安全团队的人走了,不代表AI停下来了.它只会跑得更快,但刹车可能更松了. 下周的印度AI峰会会是一个观察窗口–看看这些CEO们除了说"我们非常重视安全"之外,能不能拿出点真东西. 信息来源:BBC News,Al Jazeera,Business Insider,Financial Express
