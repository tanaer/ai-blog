把《Clean Code》《DDIA》等经典书籍转成 AI Agent 技能，代码质量提升一个档次

刚发现一个有意思的项目：把经典开发书籍转成 AI Agent 技能 📚 谁能用？ 用 AI Agent 写代码的。问题是：AI 写的代码往往是"能跑就行"，缺乏最佳实践。它不懂 Clean Code 的命名规范、DDIA 的分布式设计、Effective Java 的对象创建原则。 这个项目把 16 本经典书籍的知识蒸馏成结构化技能，让 AI 按书中的原则干活。 用法： # 安装单个技能 npx @booklib/skills add clean-code-reviewer # 安装全部技能 npx @booklib/skills add --all 有哪些技能？ 书籍 技能名 用途 Clean Code clean-code-reviewer 代码审查，C1-C5/G1-G36 启发式规则 Designing Data-Intensive Applications data-intensive-patterns 分布式系统设计 Effective Java effective-java Java 最佳实践 Effective Kotlin effective-kotlin Kotlin 安全/可读性 Domain-Driven Design domain-driven-design 战略/战术模式 Microservices Patterns microservices-patterns 微服务拆分/Saga/CQRS Refactoring UI refactoring-ui UI 视觉层次/排版 System Design Interview system-design-interview 系统设计/扩展性 还有 Effective Python、Kotlin in Action、Storytelling with Data 等。 ...
