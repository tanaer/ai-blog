Google UCP让AI帮你下单了:电商的游戏规则正在被重写

一句话概括 Google 搞了个叫 UCP(Universal Commerce Protocol)的开放协议,让 AI Agent 可以直接帮你在搜索结果里完成「发现商品→加购→结账→追踪物流」的全流程.上周已经在 Google AI Mode 里正式上线,Etsy 和 Wayfair 是第一批吃螃蟹的. 听起来像是「又一个 Google 协议」?别急,这次的影响可能比你想的大得多. 发生了什么 2026年1月,Google 联合 Shopify,Etsy,Wayfair,Target,Walmart 等电商巨头发布了 UCP 协议.2月12日,UCP 驱动的结账功能在 Google AI Mode(搜索和 Gemini App)中正式上线. 实际效果是这样的:你在 Google AI Mode 里搜「适合小户型的书架」,AI 不只是给你推荐链接,而是直接展示商品详情,旁边有个「Buy」按钮.点击后,用 Google Wallet 里存的支付方式和地址,几步就能完成购买–全程不用离开搜索页面. SEO 圈子已经炸了.Glenn Gabe 在 X 上发了实测截图:「这对电商,对 Google,对 AI 搜索来说都是大事.」 UCP 到底是什么 别被「协议」这个词吓到,UCP 的核心逻辑非常清晰: 三个核心能力: Identity Linking:基于 OAuth 2.0,让 AI Agent 代表用户在商家网站执行操作 Checkout:创建购物会话,管理购物车,计算总价,选择支付和配送方式,完成交易 Order Management:追踪物流,发起退货,更新配送状态 两个扩展机制: Extensions:可选功能模块,比如折扣,优惠券,会员积分 Services:通信层,支持 REST API 和 MCP(Model Context Protocol) 重点来了–UCP 兼容 MCP.这意味着任何支持 MCP 的 AI Agent,理论上都能接入 UCP 生态.不只是 Google 的 Gemini,Claude,GPT,开源 Agent 都有可能. ...
