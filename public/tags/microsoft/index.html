Microsoft 365 Copilot 悄悄更新:打开本地文件自动上传到云端 AI

如果你用 Microsoft 365 Copilot 作为 Android 或 iOS 的默认文档查看器,你可能刚刚被 Microsoft 「惊喜」了一下. 打开本地文件现在会自动上传到 OneDrive 并由 AI 处理–不管你愿不愿意. 发生了什么? Windows Latest 报道,Microsoft 在最近的更新中悄悄改变了 Microsoft 365 Copilot 移动端的行为. 以前: 点击 WhatsApp 里的 .docx 附件 Microsoft 365 Copilot 打开文档 你阅读文档 现在: 2. Microsoft 365 Copilot 上传文件到 OneDrive 3. Copilot 尝试「分析」文档 4. 如果 AI 能理解,你可以在聊天界面看摘要 5. 如果 AI 理解不了,你什么都看不到 官方说法 Microsoft 的提示信息是: “Opening documents from your device now starts with Copilot chat.” 翻译:「从设备打开文档现在从 Copilot 聊天开始.」 ...
