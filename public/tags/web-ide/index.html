Claude Code Open:开源版 Claude Code,带 Web IDE 和多 Agent 系统

刚发现一个猛的开源项目:Claude Code Open ️ 谁能用? 想要开源版 Claude Code 的.官方 Claude Code 是闭源的,这个项目基于公开 API 和类型定义逆向工程实现,完全 MIT 开源. 功能: CLI + Web IDE:Monaco 编辑器,文件树,AI 增强编辑 37+ 内置工具:文件操作,搜索,执行,Web 访问,任务管理,浏览器自动化 Blueprint 多 Agent:智能规划器 + 领导 Agent + 自主 Worker + 任务队列 + 质量审查 定时任务守护进程:自然语言调度,“每天 9 点代码审查” 自演化:AI 可以修改自己的源代码,热重载 一键安装 Windows: irm https://raw.githubusercontent.com/kill136/claude-code-open/private_web_ui/install.ps1 | iex macOS / Linux: curl -fsSL https://raw.githubusercontent.com/kill136/claude-code-open/private_web_ui/install.sh | bash 安装后双击桌面快捷方式,浏览器自动打开 http://localhost:3456. ## 有什么特别的? - **Web UI IDE**: Monaco 编辑器,VS Code 风格文件树,AI 悬浮提示 - **Blueprint 多 Agent**: 复杂任务自动分解到多个 Agent 并行执行 - **定时任务**: 自然语言配置时间,文件监控触发,多渠道通知 - **自演化**: AI 修改自己源码 + TypeScript 编译检查 + 热重载 - **检查点回滚**: 文件快照管理 + 会话时间旅行 - **MCP 协议**: 完整实现,支持 stdio,HTTP,SSE 传输 - **代理服务器**: 共享 Claude 订阅到其他设备 # CLI 模式 npm run dev # Web UI 模式 npm run web # 代理服务器模式(共享订阅) npm run proxy # 飞书机器人 npm run feishu # 微信机器人 npm run wechat ## 试试看 一键安装后设置 API Key: export ANTHROPIC_API_KEY=your-api-key GitHub: https://github.com/kill136/claude-code-open Demo: http://voicegpt.site:3456/ ## 时效性验证 - **HN 提交时间**: 1 分钟前(Show HN: Claude Code Open – AI Coding Platform with Web IDE and Agents) - **搜索引擎最早时间**: HN 首发 1 分钟前,GitHub 仓库为新发布 - **结论**: 最早时间 ≤ 2 小时,符合新鲜热点标准
