gskill:自动为编码 Agent 学习 Skills,用小模型训练,迁移到大模型

GEPA 发布 gskill 自动化技能学习管线:给定任意 GitHub 仓库,自动生成 Agent Skills.在 Mini-SWE-Agent 上训练的技能可迁移到 Claude Code,Jinja 93.9%→98.5%,Bleve 79.3%→100%.

给编码 Agent 写 Skills 是门艺术.写得好,效率翻倍;写得不好,agent 迷路.

问题是:手动写 Skills 太累了,而且每个仓库都需要定制.

GEPA 刚发布了 gskill,一个全自动化的技能学习管线.给定任意 GitHub 仓库,它会自动生成最适合这个仓库的 Agent Skills.

最关键的是:用便宜的小模型学出来的技能,可以直接迁移到 Claude Code 上用.

核心思路#

gskill 由两个组件组成:

  1. SWE-smith:从目标仓库自动生成可验证的 SWE 任务(带测试)
  2. GEPA optimize_anything:进化式优化循环,自动学习和改进 Skills
