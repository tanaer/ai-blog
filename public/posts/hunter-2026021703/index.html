Claude Cowork登陆Windows:Anthropic用10天造了一个要干掉SaaS的桌面AI Agent

Anthropic的Claude Cowork从macOS扩展到Windows,用Claude Opus 4.6驱动的桌面AI Agent能自主操作文件,生成文档,执行多步任务.引发2850亿美元软件股抛售,这玩意到底有多猛?
2026年2月17日 · 1 分钟 · 70 字 · AI 极客前沿
⚡ 快捷操作 🚀 发到 X

上周,Anthropic把Claude Cowork正式带到了Windows.这意味着全球70%的桌面用户现在都能用上这个"AI桌面代理"了.

听起来像又一个AI助手?不是.这东西从根本上改变了AI和你电脑的关系.

不是聊天机器人,是真的在你电脑上干活#

传统的AI助手,包括ChatGPT和之前的Claude,本质上都是"你问我答"–你描述问题,它给建议,然后你自己动手.

Cowork不一样.你说"把我的研究文件夹按主题整理,然后生成一份摘要文档",它会:

  1. 读取你的文件
  2. 理解每个文件的内容
  3. 创建文件夹结构
  4. 移动文件
  5. 生成一份格式完美的摘要文档

全程你可以去泡杯咖啡.

技术规格:硬核拆解#

Cowork底层跑的是Claude Opus 4.6,带beta版100万token上下文窗口.这意味着它能一次性"看到"你项目里几十个文件的全部内容,而不是像其他AI那样只能处理一小段.

核心能力:

  • 本地文件系统访问:直接操作你电脑上的文件,数据不需要上传到云端, 多步骤任务执行:拆解复杂任务为子任务,并行执行, 专业文档生成:Word,Excel,PowerPoint,PDF,带公式,样式,目录, MCP连接器:通过Model Context Protocol接入外部服务, 插件系统:1月30日上线的11个开源插件,覆盖销售,法务,财务,营销,数据分析,开发

价格:Pro $20/月,Max $100/月起,免费用户用不了.

最疯狂的细节:这玩意是Claude Code用10天写出来的#

Anthropic官方说法:Cowork的全部代码是Claude Code自己写的,团队从零到发布只用了大约10天.

让一个AI写另一个AI产品,而且是面向消费者的桌面应用–这在软件工程史上算是一个标志性事件.它证明了AI写生产级代码的能力已经不是实验室玩具了.

2850亿美元的恐慌#

Bloomberg报道,Cowork发布后软件股出现了2850亿美元的集体抛售.投资者的逻辑很简单:如果一个AI代理能自主完成文件管理,数据分析,文档生成这些工作,那Notion,Asana,Airtable这类工具的护城河在哪?

这不是杞人忧天.想想看:

  • 你用Notion整理笔记?Cowork直接在你本地文件系统上做, 你用Excel处理数据?Cowork帮你从原始数据到分析报告一步到位, 你用PowerPoint做演示?描述一下要求,Cowork生成一个带样式的.pptx

SaaS的核心价值–帮你组织和处理信息–正在被AI Agent侵蚀.

对比微软Copilot和Google Gemini#

三者定位其实不同:

Microsoft Copilot:嵌入Office 365,帮你在Word里写得更快,在Excel里分析更准.本质是增强现有工具.

Google Gemini:深度集成Google Workspace,擅长搜索和Gmail.本质是连接Google生态.

Claude Cowork:独立运行,直接操作你的文件系统,不依赖任何生态.本质是替代工作流本身.

Copilot让你的Word更好用,Cowork让你不需要打开Word.

插件生态是关键#

1月30日上线的插件系统才是Cowork真正的杀手锏.11个官方开源插件涵盖了几乎所有白领工种:

  • 销售助手:CRM数据分析,客户跟进, 法律助手:合同审查,条款分析, 财务助手:报表生成,预算追踪, 开发助手:代码审查,文档生成

而且这些插件是开源的,意味着社区可以在此基础上扩展.按照OpenClaw的Skill生态发展速度来推算,Cowork的插件数量可能在几个月内爆发式增长.

我的判断#

Claude Cowork代表了AI从"对话"到"行动"的转变.这是2026年AI领域最重要的产品形态创新之一.

但现阶段它更适合个人知识工作者–自由职业者,研究人员,内容创作者.企业级用户还需要等权限管理,审计日志等功能完善.

如果你是付费Claude用户,现在就可以试试Cowork模式.打开Claude Desktop应用,切换到Cowork标签,给它一个复杂任务,看它怎么拆解和执行.

这种"描述目标→AI自主完成"的工作方式,一旦用过就回不去了.

Claude Cowork macOS版1月发布,Windows版2月10日上线.需要Claude Pro及以上订阅.
